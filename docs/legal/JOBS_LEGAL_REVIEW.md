> ⚠️ **나는 변호사가 아니다.** 이 문서는 사내 1차 법무 검토다. 정식 상용 출시(유료 연결·수수료) 전에는 **고용노동부 신고 실무 + 노동·행정 전문 변호사 검토**가 필요하다. 단, 무료 정보형 단계의 합법성 판단과 조문 근거는 아래대로 신뢰 가능하다.

## 0. 결론 한 줄 (TL;DR)

원티드·잡코리아·링크드인·자소설닷컴이 합법인 이유는 그들이 **"직업소개(알선)"가 아니라 "직업정보제공"**을 하기 때문이다. 직업안정법은 이 둘을 **행위로** 구분한다 — **불특정 다수에게 정보를 제공만** 하면 직업정보제공사업(신고제), **특정 구인자와 특정 구직자를 짝지어 고용계약 성립을 알선**하면 직업소개사업(등록/허가제)이다. WelKor의 "출신국×비자×직무 맞춤 큐레이션 + 외부 딥링크"는 **알선이 아니라 정보제공**이므로 합법이나, **사업으로 직업정보를 제공하면 직업안정법 제23조 직업정보제공사업 신고 대상**이다(신고는 무료·간단·온라인). E-9는 별개 — 고용허가제(정부 독점)라 민간 알선 자체가 형사처벌.

## 1. 회장 B질문 정면 답변 — "원티드·잡코리아·링크드인은 왜 합법인가"

### 1-1. 직업안정법은 4개의 행위를 구분한다 (제2조의2 정의)

| 행위 | 법정 정의 (직업안정법 제2조의2) | 규제 수준 |
|------|-------------------------------|----------|
| **직업정보제공** | 8호: "신문, 잡지, 그 밖의 간행물 또는 유선·무선방송이나 **컴퓨터통신 등으로 구인·구직 정보 등 직업정보를 제공하는 사업**" | **신고제** (제23조) |
| **직업소개** | 2호: "구인 또는 구직의 신청을 받아 구직자 또는 구인자를 **탐색하거나 구직자를 모집하여 구인자와 구직자 간에 고용계약이 성립되도록 알선하는 것**" | 무료=신고(제18조) / 유료=**등록제**(제19조) |
| **근로자모집** | 6호: 고용하려는 자가 취업하려는 사람에게 피고용인이 되도록 권유 (= 채용 주체 본인이 함) | 원칙 자유(제28조), 국외모집만 신고(제30조) |
| **근로자공급** | 7호: "공급계약에 따라 근로자를 타인에게 사용하게 하는 사업"(파견 제외) | **허가제**(제33조) — 노동조합만 |

### 1-2. 결정적 경계 — "정보제공"과 "알선"을 가르는 법정 기준

고용노동부 행정해석(직업안정법 제25조·시행령 제28조 관련 실무 안내)은 둘을 이렇게 가른다:

> - **직업소개사업**은 구인자와 구직자를 **직접 연결하여 알선**하는 것
> - **직업정보제공사업**은 **불특정다수의 잠재적 구인자와 구직자에게 직업정보를 제공만** 하고, 구인·구직자가 **스스로** 구인 또는 구직하도록 하는 것
> - **직업정보제공사업자는 직업소개사업의 등록을 별도로 하지 아니하고는 구인·구직자 알선, 소개 등 직업소개사업 업무를 할 수 없음**

→ **핵심 변수 = "누가 매칭의 마지막 한 걸음을 밟는가."** 정보를 보고 **유저 본인이** 지원·연락하면 정보제공(합법). 플랫폼이 **특정인을 특정 회사에 짝지어 연결·추천**하면 알선(직업소개).

### 1-3. 그래서 원티드·잡코리아·링크드인·자소설닷컴은?

이들은 모두 **직업정보제공사업자**다(직업안정법 제23조 신고). 채용공고를 게재·검색·필터·알림하지만, 마지막 지원 행위는 **구직자 본인이** 한다. 잡코리아·사람인·원티드는 직업정보제공사업 신고 사업자이며(원티드는 일부 헤드헌팅 라인에서 별도로 유료직업소개 등록을 병행), 링크드인은 글로벌 직업정보 플랫폼이다.

**Alpha의 앞선 단순화("원티드식 정보제공까지 불법")는 부정확했다.** 원티드식 정보제공은 합법이다 — 다만 **무신고로 하면 위법**이고, "추천"이 알선으로 넘어가면 등록 대상이 된다. 경계는 "정보제공이냐 알선이냐"이지 "맞춤이냐 아니냐"가 아니다. **맞춤·추천 알고리즘 자체는 정보제공의 범주 안에서 합법**이다(아래 §3 트리거 참조).

## 2. 웰코 합법 스펙트럼 — 4단계 + 각 요건·벌칙·외국인 특칙

### ① 단순 외부 링크 (현행 jobs.ts)
- **내용:** 정적 외부 사이트 북마크 12개. 구인·구직 정보를 **직접 보유·게재하지 않고** 타 사이트로 보내기만 함.
- **법적 판정:** 직업안정법 규제 밖. 신고 불요. (단순 하이퍼링크 = "직업정보를 제공하는 사업"으로 보기 어려움)
- **문제:** 합법이지만 가치 0(회장 지적). 죽은 링크(Contact Korea=KOTRA GTC 통합으로 폐지) 포함.

### ② 직업정보제공사업 (제23조 신고제) ★ 웰코의 합법 최대치
- **내용:** 채용공고를 **플랫폼이 직접 게재·DB화**, 검색·필터·**맞춤추천**·알림. 구직자가 보고 본인이 외부에서 지원.
- **요건 (제23조, 시행령):**
  - **고용노동부장관에게 신고** (고용24/정부24 온라인, 처리 통상 신속). 신고 사항·절차는 시행령.
  - 제출서류(실무): 신규신고서, 사업자등록증, (법인) 등기·정관, **2년치 사업계획서**(사이트 주소·제공정보·대상직종·이용방법 명시).
  - 신고일부터 **6개월 이내 사업 개시** 의무.
  - **신고번호를 매체에 표시**(시행령 제28조 5호).
- **준수사항 (시행령 제28조 — 현행, 시행 2026.3.23. 대통령령 제36220호) — 정보제공의 합법 경계선:**
  1. 구인자 신원(업체명·성명·사업자등록증 등) 불명확하거나 연락처가 사서함 등인 **불명확 구인광고 게재 금지**
  2. 광고에 **구인·구직자**의 주소·전화번호를 기재하고, **직업정보제공사업자의** 주소·전화번호는 기재하지 말 것 (= 플랫폼이 연락 창구가 되지 말 것)
  3. **"(무료)취업상담"·"취업추천"·"취업지원" 등의 표현 사용 금지** ← 알선처럼 보이는 표현 금지
  4. **구직자 이력서 발송 대행 금지, 구직자에게 취업추천서 발부 금지** ← 알선 행위 금지
  5. **신고번호 표시**
  6. 최저임금 미달 구인정보·성매매알선 금지행위 업소 구인광고 게재 금지
- **벌칙:** 미신고 직업정보제공사업 영위 = 직업안정법 제48조 위반(형사처벌 대상). 시행령 준수사항 위반 시 사업정지 처분(실제 판례 다수: 대법원 2020두51587 등).
- **외국인 특칙:** 외국인등록번호·여권번호는 개인정보보호법상 **고유식별정보** → 별도 동의·암호화·국외이전 고지 필요. **직업정보제공 단계에서는 외국인등록번호를 수집할 이유 자체가 없다**(공고 보고 외부 지원하므로). 수집하지 않는 설계가 정답.

### ③ 직업소개사업 (제19조 유료=등록제 / 제18조 무료=신고제)
- **내용:** 특정 구직자를 특정 구인자에게 **연결·추천하여 고용계약 성립을 알선.** 헤드헌팅, 지원대행, "이 후보 뽑으세요" 추천.
- **유료직업소개 등록 요건 (제19조 + 시행령 제21조 + 시행규칙 제17조):**
  - 관할 **구청 등록**(국내) / 고용노동부장관 등록(국외). 결격사유 조회 + **사무실 현장조사**, 처리 15일.
  - **전용 사무실**(독립 공간) + 자격 갖춘 **직업상담원 1명 이상 고용**(제22조).
  - **둘 이상 사업소 금지** 원칙(제19조 2항).
  - 고용노동부장관 **고시 요금 외 금품 수령 금지**(제19조 3항, 위반 시 과태료 1천만원 — 제50조). 단 고급·전문인력 소개는 당사자 합의 요금 가능.
  - **선급금 수령 금지**(제21조의2), **명의대여 금지**(제21조), **연소자 소개 제한**(제21조의3), **손해배상책임 보장**(보증보험·공제 — 제34조의2), **장부 비치**(제39조).
  - **겸업 금지**(제26조): 식품접객업·숙박업 경영자는 직업소개사업 불가.
- **벌칙:** **무등록 유료직업소개 = 제47조 5년 이하 징역 또는 5천만원 이하 벌금**(2014.5.20. 개정으로 2천→5천만원 상향). 강남구청 안내(2026-03-12)도 "5년 이하 징역 또는 5천만원 이하 벌금" 명시.
- **외국인 특칙:** **국외 유료직업소개**(해외→한국 또는 한국→해외 취업 연결)는 별도 등록(고용노동부장관). 해외이주알선업과 별개로 직업안정법 제19조 국외등록 필요(법령해석례 확인).

### ④ E-9 등 비전문취업 알선 금지선 (외국인고용법 = 절대 금지)
- **내용:** E-9(비전문취업)·E-10(선원) 외국인의 채용 연결.
- **법적 판정:** **민간이 손댈 수 없음.** 외국인근로자의 고용 등에 관한 법률상 **고용허가제(EPS)** = 사용자가 직업안정기관의 장에게 고용허가 신청, 정부(EPS)가 송출국 정부와 MOU로 운영하는 **정부 독점 채널**(외국인고용법 제8조·제9조). 민간이 E-9 구인·구직을 알선하면 직업안정법 위반 + 외국인고용법 위반 + (수단에 따라) 직업안정법 제46조(폭행·성매매 목적 알선 = 7년 이하 징역) 가능.
- **웰코 처리:** **MVP 완전 제외 확정.** EPS는 "정보 안내(링크)"로만, 클릭형 채용보드에서 분리.

## 3. 핵심 판정 — 웰코 "맞춤 큐레이션 + 외부 딥링크"는 신고가 필요한가?

### 판정: 행위 자체는 합법(알선 아님). 단, **사업으로 직업정보를 제공하면 직업안정법 제23조 신고가 필요하다.**

| 웰코가 하는 일 | 신고 필요 여부 | 근거 |
|---------------|--------------|------|
| (a) 외부 사이트로 **단순 링크만** (정보 직접 보유·게재 X) | **신고 불요** | "직업정보를 제공하는 사업" 미해당. 현행 jobs.ts가 이 수준 |
| (b) 비자·출신국별 **가이드/큐레이션** + 외부 딥링크 (공고 DB 직접 보유 X, 큐레이션 = 콘텐츠) | **회색지대 → 신고 권고(안전)** | 직업정보를 "제공"하는 외관 강해짐. 무신고 리스크 회피 위해 신고가 안전 |
| (c) 채용공고를 **플랫폼이 직접 게재·검색·맞춤추천** | **신고 필요 (제23조)** | 명백한 직업정보제공사업 |

→ **"맞춤 추천"은 ②③ 경계가 아니라 ①② 경계 변수다.** 맞춤·추천 알고리즘은 **직업정보제공의 범위 안에서 합법**이다(쿠팡·넷플릭스식 추천처럼). 추천한다고 알선이 되지 않는다. **알선이 되는 트리거는 "맞춤"이 아니라 "특정 매칭 + 개입"이다.**

### "맞춤 추천 → 알선(직업소개)"으로 넘어가는 트리거 (이걸 건드리면 등록 대상)

1. **특정 후보를 특정 기업에 1:1로 연결·추천** (불특정 다수 공개 게시가 아니라, "이 사람을 이 회사에") → 알선
2. **지원/이력서 발송을 플랫폼이 대행** → 시행령 제28조 4호 위반 + 알선
3. **취업추천서 발부, "취업추천"·"취업상담"·"취업지원" 표현 사용** → 시행령 제28조 3·4호 직접 위반
4. **합격 여부·고용계약 성립에 플랫폼이 관여**(면접 주선·조율·합격 통보 중개) → 알선
5. **수수료(소개비)를 받음** → 유료직업소개 = 제19조 등록 의무(무등록 시 5년/5천만원)
6. **플랫폼이 연락 창구가 됨**(공고에 플랫폼 연락처 기재) → 시행령 제28조 2호 위반

**안전 규칙:** 웰코는 "정보를 보여주고 유저를 외부로 내보내는" 지점에서 **반드시 멈춘다.** 매칭의 마지막 한 걸음(지원·연락·합격)은 100% 유저 본인 또는 외부 사이트가 밟는다.

### 확인된 사실 (1차 출처)
- 직업안정법 제2조의2(정의)·제18·19·22·23·25·26·33·46·47·48·50조 = yeslaw 법령 전문으로 조문 확인.

- **시행령 제28조(직업정보제공사업자 준수사항) = 현행, 시행 2026.3.23. 대통령령 제36220호** = LBOX 법령으로 6개 호 전문 확인. (회장 요청 "2024~2026 개정 확인" 충족 — 2022.12.27. 개정분이 현행 반영됨.)

- E-9 고용허가제(정부 독점) = 외국인고용법 제8·9조 + EPS 안내 확인.

### 미확인 / 변호사 확인 필요
1. **제48조(미신고 직업정보제공·무료직업소개) 현행 벌금 정확액** — 검색 스니펫상 변동 명시 없어 "1년 이하 징역 또는 종전 500만원~상향분" 범위로 추정. 정확 액수는 law.go.kr 제48조 조문 직접 확인 필요. (결론에는 영향 없음 — 미신고는 어느 쪽이든 형사처벌 대상.)
2. **(b) 큐레이션+딥링크가 "직업정보 제공 사업"에 해당하는지의 정확한 경계** — 행정해석/판례가 단순링크와 직접게재 사이 회색지대를 명확히 그은 바 없음. 사업화·트래픽 본격화 시점에 관할 고용센터 유권해석 또는 변호사 확인 권고. (안전책 = 신고하면 끝.)
3. **신고 주체 적격** — 웰코 사업자등록·법인격 미정 상태. 신고는 사업자등록 후 가능. 무료 베타·미등록 단계에서 (a)단순링크는 신고 없이 가능하나, (b)/(c)로 갈수록 사업자등록+신고 선행 필요.
4. 시행령 제28조는 "직업정보제공사업자"(=신고한 자)의 준수사항이나, **무신고로 정보제공하면서 이를 위반하면 미신고(제48조) + 준수사항 위반 이중 리스크.** 신고를 먼저 하는 것이 모든 면에서 안전.

## 6. FOUNDATION.md §4 정정 권고

현행 FOUNDATION.md §4의 "구직 정보 게시판(무료) ✅ 합법 / 유료화 전 직업소개사업 신고"는 **부정확.** 정정:
- 무료 단순 링크 = 신고 불요(✅).
- **무료라도 직업정보를 직접 게재·제공하는 사업이면 "직업소개사업 신고"가 아니라 "직업정보제공사업 신고(제23조)"가 맞음.** 유료/무료가 아니라 "정보제공이냐 소개(알선)이냐"가 신고/등록을 가른다.
- "유료화 전 직업소개사업 신고"라는 표현은 직업소개(알선)로 오인 → 등록제(5년/5천만원 리스크)로 잘못 끌고 감. 웰코는 알선 안 하므로 직업소개 등록 대상 아님.
