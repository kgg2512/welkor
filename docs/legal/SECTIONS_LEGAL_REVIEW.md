# WelKor 섹션별 법률 리스크 검토

**작성:** CLO (g2-clo) | **날짜:** 2026-06-26 | **대상 섹션:** housing, tax, finance, study, community, explore, listings
**참조법령:** 공인중개사법(2026.2.15. 시행 법률 제21024호), 세무사법, 자본시장법, 외국환거래법, 표시광고법, 개인정보보호법
**주의:** CLO는 변호사가 아닙니다. 본 검토는 사내 1차 법무이며, 상용 출시 전 반드시 법무법인 정식 자문이 필요합니다.

## 리스크 등급 요약 (높음순)

| 순위 | 섹션 | 리스크 등급 | 핵심 리스크 한 줄 |
|------|------|------------|-----------------|
| 1 | tax (세무) | 높음 | 세무사법상 세무상담/세무대리 경계 — 구체 세율·조약 적용 판단 제공 시 무자격 세무대리 위반 소지 |
| 2 | housing / listings (부동산) | 중간-높음 | 공인중개사법상 "알선" 정의 — 특정 매물과 특정 임차인을 연결하는 순간 중개행위 해당 가능 |
| 3 | finance (금융) | 중간 | 자본시장법 투자자문업 미등록 리스크 / 대출 안내의 대부업법 저촉 가능성 |
| 4 | study (유학) | 중간 | 외국교육기관 유치·알선 등에 관한 규정(교육부 고시) 미준수 시 행정지도 |
| 5 | community | 낮음-중간 | UGC 관련 명예훼손·저작권·개인정보 — 플랫폼 면책 구조 미비 |
| 6 | explore (여행/관광) | 낮음 | 관광진흥법상 여행업 미해당 확인 필요 — 현 구조는 적법 |
| 7 | listings (현 Mock 데이터) | 낮음 | 샘플 매물임을 명시 — 표시광고법상 "예시 매물" 고지 유지 필수 |

## 1. housing (부동산) — 리스크: 중간-높음

### 1-1. 현 구현이 하는 것
- `HOUSING_PORTALS`: 네이버부동산·직방·다방·집토스 4개 포털 외부링크 제공
- `REGION_PRICES`: 원룸 시세 참고치(지역별 보증금·월세 범위)
- `SHORT_STAY`: 에어비앤비·부킹닷컴·아고다 단기숙박 링크
- `LISTINGS`: 샘플 매물 4개(외국인 친화, 보증금·월세 금액 포함) — 현재 Mock
- UI 문구: "WelKor는 공인중개사가 아닙니다 — 중개·보수 수취를 하지 않습니다"

### 1-2. 법적 리스크 분석

**공인중개사법 제2조 제1호 (2026년 2월 시행 기준, law.go.kr 직접 확인):**
> "중개"란 제3조에 따른 중개대상물에 대하여 거래당사자간의 매매·교환·임대차 그 밖의 권리의 득실변경에 관한 행위를 **알선**하는 것

**제9조:** 중개업을 하려면 등록관청에 중개사무소 개설등록 필요 (공인중개사 자격 보유자만)
**제48조:** 미등록 중개업 = 3년 이하 징역 또는 3천만원 이하 벌금

**현 구현 위험 지점:**
1. **단순 포털 링크 → 합법.** 포털에서 유저가 알아서 검색·임차인 매칭: 알선행위 없음. 네이버부동산·직방·다방이 합법으로 운영되는 것과 동일 구조.
2. **시세 참고치 제공 → 합법.** 통계적 정보 제공은 알선이 아님. 다만 특정 매물의 적정 임대료를 "이 집은 얼마가 맞다"고 판단해 주면 상담에 가까워짐 → 금지.
3. **`LISTINGS` 샘플 매물 → 주의 필요.** 현재 Mock 데이터이므로 문제 없으나, **실매물 연결 시 리스크 발생 구간:**
   - 유저가 매물을 보고 WelKor 내에서 공인중개사에게 연결 요청 → WelKor가 중간에서 "이 집-이 사람" 매칭 행위를 하면 알선 해당
   - 안전선: 유저를 공인중개사 프로필 페이지로 보내는 것까지만 (이후 계약은 당사자 직접)
4. **확정일자·전세사기 가이드 제공 → 합법.** 공공정보·체크리스트 안내는 알선이 아님. 단 "이 집 계약해도 됩니다"라는 개별 안전 판단 제공 금지.

**공인중개사법 제18조의2·3 (인터넷 표시·광고):**
- 매물 정보를 인터넷에 광고할 때 공인중개사만 게시 가능 (직접 중개 목적 광고)
- WelKor가 게시하는 것이 "중개 목적 광고"로 해석될 경우 위반
- 안전선: 샘플/예시임을 명확히 표시 + 실매물은 공인중개사 입점자가 직접 게시 (WelKor는 플랫폼만 제공)

### 1-3. 합법 모델
- WelKor = 부동산 정보 포털 + 공인중개사 마켓플레이스 (직방·다방 구조)
- 포털 딥링크 제공 + 시세 통계정보 + 공인중개사 입점 디렉토리만
- WelKor 플랫폼은 매물 게시 주체가 아닌 입점 공인중개사가 직접 매물 게시 + 거래 주체
- WelKor는 중개보수를 절대 수취 안 함

### 1-4. 필수 고지문구

**현 ko.json 문구 분석:** housing.subtitle = "WelKor는 정보만 제공합니다 — 공인중개사가 아니며 중개·보수 수취를 하지 않습니다" — **적절하나 한 가지 추가 필요.**

**추가 필수 고지 (실매물 단계 도달 시):**
```
이 플랫폼의 매물은 입점 공인중개사가 직접 게시합니다. 
WelKor는 중개대상물에 대한 알선·계약 체결에 관여하지 않습니다.
보증금 지급 전 반드시 등기부등본 확인과 공인중개사의 확인·설명서를 받으세요.
(공인중개사법 제25조)
```

## 2. tax (세무) — 리스크: 높음

### 2-1. 현 구현이 하는 것
- `TAX_GUIDES` 4개 항목:
  - t1: 연말정산 개요 (직원 대상 연 1회 정산 설명)
  - t2: 외국인 단일세율 특례 — "단일세율 선택 가능, 유리한지 비교 필요"
  - t3: 조세조약·교사 면세 — "일부 교사는 최대 2년 면세, 국적·계약 따라 다름"
  - t4: 유학생 아르바이트 소득 — "원천징수 대상, 환급 가능"
- UI 문구: "가이드만 제공 — 신고는 세무사가 수행", "일반 정보이며 세무 자문이 아닙니다"

### 2-2. 법적 리스크 분석

**세무사법 제2조 (세무대리의 정의, 주석 포함 법령 지식 기반):**
- 세무사·공인회계사·변호사 아닌 자의 세무대리 금지
- 세무대리 = 조세에 관한 신고·신청·청구·불복·세무 조정·확인·상담 등
- **제22조:** 무자격 세무대리 행위 = 1년 이하 징역 또는 1천만원 이하 벌금

**"세무상담"의 경계:**
- **합법 (일반 정보):** "연말정산이 무엇인지", "단일세율 제도가 있다", "조세조약 면세 제도 존재", "연금 환급 제도 존재" 등 **제도의 존재와 개요** 설명
- **위험 (세무상담/대리 경계):** "당신의 국적·계약에서는 면세가 됩니다", "당신은 단일세율이 유리합니다", "이 금액의 환급을 받을 수 있습니다" 등 **특정인에 대한 적용 판단**

**현 구현 위험 지점:**
- t2: "유리한지는 소득에 따라 다르므로 비교가 필요합니다" → 비교 방법 안내 자체는 정보. 단, "당신은 [X]가 유리"라는 판단 제공 시 위반 소지
- t3: "국적·계약에 따라 다르므로 신고 전 확인하세요" → 적절. 단, 국가별 조세조약 내용을 상세 해석해 "미국인은 2년 면세"처럼 특정 적용 판단 제공 시 경계 침범
- 세무사 연결 버튼(connectAccountant) → 이것은 정보 제공이 아닌 전문가 연결로, **합법이며 오히려 이 버튼이 있어야 안전**

**삼쩜삼 분쟁 참고:** 2022년 국세청이 삼쩜삼(자비스앤빌런즈)에 세무사법 위반 여부 검토 → 세무대리 논란. 웰코는 삼쩜삼보다 훨씬 소극적(계산·신고 없음)이나, 조세조약 적용 판단은 동일한 경계선상에 있음.

### 2-3. 합법 모델
- **허용:** 제도 소개 + 체크리스트 + "세무사에게 물어보세요" 연결
- **금지:** 특정 유저에 대한 세율 계산, 면세 적용 여부 판단, 환급액 추산
- 현 TAX_GUIDES 수준은 합법이나, 향후 기능 추가 시 "계산기/판단기" 방향은 법무법인 자문 선행 필수

### 2-4. 필수 고지문구

**현 ko.json 문구:** "일반 정보이며 세무 자문이 아닙니다. 본인의 경우는 면허 전문가에게 확인하세요." — **충분하나 위치와 크기가 중요.** 각 가이드 카드 상단(접힘 전)에 표시해야 효력.

**추가 고지 권고 (조세조약·면세 항목에 필수):**
```
이 정보는 제도의 개요를 설명하는 일반 정보입니다.
귀하의 국적·계약·소득에 따른 세금 적용 여부 판단은 세무사(세무사법 제2조)만이 
수행할 수 있습니다. WelKor는 세무대리를 하지 않습니다.
```

## 3. finance (금융) — 리스크: 중간

### 3-1. 현 구현이 하는 것
- 4개 카테고리: 은행계좌 개설 안내, 체크·신용카드 안내, 대출 안내, 해외송금 안내
- 각 카테고리별 기관 외부링크 + 이용 자격 설명
- 금융감독원 공식 링크 병기
- 문구: "WelKor는 금융기관·대부업체·금융자문이 아니며 어떤 금융상품도 제공하지 않습니다"

### 3-2. 법적 리스크 분석

**자본시장법 제17조 (투자자문업):**
- 투자자문업 = 금융투자상품의 가치·투자 판단에 관해 자문하는 행위 (등록 필수)
- 현 구현: 은행계좌·체크카드·대출·송금 — **투자상품이 아님** → 투자자문업 해당 없음
- 리스크 없음 (현재 주식·펀드·암호화폐 등 투자 상품 없음)
- **주의:** 향후 투자 관련 정보(주식계좌 개설, 펀드 추천 등) 추가 시 투자자문업 등록 또는 면책 구조 검토 필요

**외국환거래법:**
- 해외송금 서비스 자체를 WelKor가 제공하는 것이 아니라 Wise·Sentbe 등 **등록 업체에 링크** → 문제 없음
- "등록 업체만 이용하라"는 안내는 오히려 컴플라이언스 보조 기능

**대부업법:**
- 대출 연결: 은행(등록 금융기관)으로의 링크만 → 문제 없음
- "무허가 사금융 절대 피하라"는 현 문구(loans 카테고리)는 적절하며 법적 리스크 차단

**표시광고법 제3조:**
- 금융상품 비교 정보 제공 시 허위·과장 광고 금지
- 현 구현: "외국인 지원 상이"(Toss Bank 주석) 등 조건부 표현 사용 → 적절
- "투명한 수수료"(Wise 설명) → 사실에 부합하는 한 문제없음

### 3-3. 합법 모델
- 현 구현 구조 자체는 **합법**
- WelKor = 공공 정보 큐레이션 + 공식 기관 링크 제공자
- 금융상품 비교 플랫폼(파인더·뱅크샐러드 방식)으로 발전 시 전자금융거래법·표시광고법 추가 검토 필요

### 3-4. 필수 고지문구

**현 문구 적절.** 다음 사항 확인·유지:
```
WelKor는 금융기관·대부업체·금융자문이 아닙니다. 이 페이지의 정보는 일반 
안내이며, 자격·수수료·금리는 각 기관이 결정합니다. 반드시 허가된 기관(금융감독원 
등록 여부 확인)만 이용하세요.
```

**추가 권고 (대출 카테고리):**
```
대출은 반드시 금융감독원에 등록된 금융기관을 통해서만 받으세요.
미등록 대부업자(사금융) 이용은 불법이며 피해 보상이 어렵습니다.
(대부업법 제3조, 금융감독원 서민금융 1332)
```

## 4. study (유학) — 리스크: 중간

### 4-1. 현 구현이 하는 것
- 정부 포털(studyinkorea.go.kr) 링크
- 전공 분야 8개 소개
- 8개 실존 대학 정보 + 공식 사이트 링크 + 입학 검색 쿼리(라이브 검색으로 연결)
- D-2/D-4 비자 서류 체크리스트
- GKS 장학금·TOPIK 공식 링크
- HiKorea 공식 비자 링크
- 문구: "유학원이 아니며 대신 지원서를 작성하지 않고, 링크로 수익을 얻지 않습니다"

### 4-2. 법적 리스크 분석

**외국교육기관 유치·알선 등에 관한 규정:**
- "외국유학생 유치·알선업"에 대한 별도 등록 의무는 **현재 국내법상 명시적 규정이 없음** (확인 결과: 유학원에 대한 직접 규제 단독 법률 부재)
- 대신 **표시광고법** 적용: 대학·프로그램에 대한 허위·과장 정보 제공 금지
- **수수료·알선료 수취 시** 소비자보호법상 중개업 해당 가능 + 부당행위 처벌 리스크

**현 구현 위험 지점:**
1. **입학 정보 제공 → 합법.** 공식 사이트·정부 포털 링크만 제공, 입학 지원 대행 없음
2. **대학 정보 표시 → 주의 필요.** admissionsQuery(라이브 검색)는 stale 정보 방지 장치로 적절. 단, 대학 정보(입학 조건·등록금 등)가 변경될 경우 업데이트 의무
3. **비자 서류 체크리스트 → 합법.** "정확한 목록은 국적·공관별로 다르므로 반드시 확인하라"는 단서 존재 → 적절
4. **수수료 없음 확인 필수.** 현재 무료이므로 알선업 해당 없음. 유료화 시 재검토 필요

**행정사법 제2조 (행정사 업무 범위):**
- 비자 서류 작성·제출 대행 = 행정사 업무
- 현 구현: 셀프 가이드(본인 작성 안내)만 제공 → 합법
- "지원서를 작성하지 않습니다" 고지 유지 필수

### 4-3. 합법 모델
- 정보 제공 + 공식 링크 큐레이션 = 현 구조 합법
- 입학 대행·비자 대행·수수료 수취 완전 금지
- 유학원 입점 마켓플레이스 구조로 확장 시: 소비자보호법·표시광고법 적용 → 법무 자문 필요

### 4-4. 필수 고지문구

**현 문구 적절.** 다음 항목 점검·보강:
```
WelKor는 유학원이 아닙니다. 대학 정보·비자 요건은 각 대학·정부 기관이 정하며,
WelKor가 제공하는 정보는 참고용입니다. 입학 지원서 작성·제출·비자 서류 대행을

대학 지원 전 반드시 해당 대학 국제처와 주한 공관에서 최신 요건을 확인하세요.
```

## 종합 행동 권고 (우선순위 순)

### 즉시 조치 (런치 전 필수)

1. **[tax] 고지문구 위치 개선:** 현재 페이지 하단 disclaimer에만 있는 "세무 자문 아님" 문구를 각 가이드 카드 상단(접힘 전 노출)으로 이동. 특히 t2(단일세율), t3(조세조약) 항목에 강조 필요.

2. **[housing] 실매물 전환 계획 수립:** Mock→실매물 전환 전 공인중개사 입점 계약서 작성 + 자격 검증 절차 설계. "알선"의 경계를 넘지 않는 매칭 UX 설계 필요 (클릭 후 중개사 프로필로 직행, WelKor 중간 개입 없음).

3. **[community] 이용약관 준비:** 실 UGC 도입 전 이용약관(허위정보·저작권·개인정보 금지) + 신고 절차 + 임시조치 절차 구비.

4. **[study] 대학 정보 업데이트 정책:** admissionsQuery 라이브 검색은 적절. 단, 대학 정보(known 필드 등) 수동 업데이트 주기 정책(연 1회 이상) 수립.
