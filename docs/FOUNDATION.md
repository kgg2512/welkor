## 0. 한 줄 정의

> 한국에 막 도착한 외국인이 **비자 → 주거 → 일자리 → 세무 → 자녀학교**의 정착 전 과정을, 영어·한국어 스텝바이스텝으로 끝내고, 필요한 순간 **검증된 면허 전문가(공인중개사·세무사·행정사)**와 연결되는 통합 플랫폼.

핵심 차별화: 경쟁사는 전부 **수직 단일**(세무만/주거만/구직만). 웰코는 **비자 단계에 맞춘 수평 통합 여정 + 커뮤니티 락인.**

- **총 체류외국인 2,650,738명** (+5.2%/yr), 장기체류 204만(사상 첫 200만 돌파), 경제활동 156만(+9.1%).

- 국적 상위: 중국 35.2% · 베트남 12.1% · 미국 6.5% · 태국 6.1% · 우즈베키스탄 3.7%.

## 5. MVP 범위 (디자인보다 기능 우선 — 회장 지시)

**Phase 1 MVP = 정보 허브 + 부동산/전문가 리스팅 + 커뮤니티. 핵심 도메인 = 부동산·세무.**

핵심 기능:
1. **정착 여정 엔진(스텝바이스텝):** 유저 비자 종류 선택 → 그 단계에 맞는 체크리스트·다음 할 일 카드(비자→전입신고→은행계좌→집→세무). *웰코의 진짜 차별화 = 이 엔진.*
2. **부동산 리스팅:** 외국인 친화 매물(영문, 보증금 구조·확정일자·전세사기 경고 가이드 포함) + 공인중개사 입점.
3. **세무 가이드 + 세무사 디렉토리:** 연말정산·조세조약 환급 가이드(정보) + 검증된 세무사 연결.
4. **커뮤니티:** 국적/지역별 Q&A·후기 (락인 자산).
5. **다국어:** 영어·한국어 1차 (i18n 구조는 다국어 확장 가능하게).

비포함(후행): 비자/세무/구직 대행, 결제, 신용카드, 관리비, 국세청 연동, 모바일 앱.

## 7. 기술 스택 제안 (기능·SEO 우선, $0)

- **Next.js (App Router) + next-intl** — 다국어 + SSG/SSR. **SEO 결정적**: 외국인이 "how to get D-2 visa korea" 구글링 → 유기 유입이 핵심 성장 채널.
- **Supabase** — DB·Auth·Storage (G2 기존 스택).
- **Vercel** 배포 (또는 GitHub Pages 정적 부분).
- 디자인은 최소 컴포넌트(기능 우선) — Tailwind + 기본 UI 키트.
- 데이터 모델 우선 설계: `users(visa_type, nationality, locale)`, `listings(real_estate)`, `professionals(type, license_verified)`, `community_posts`, `journey_steps`.
